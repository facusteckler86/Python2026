## 📜 Un poco de historia
Python fue creado a finales de los años ochenta por **Guido van Rossum** en el *Stichting Mathematisch Centrum (CWI)*, en los Países Bajos. Surgió como el sucesor del lenguaje ABC, con la capacidad de manejar excepciones e interactuar con el sistema operativo Amoeba.

### 🧘 El Zen de Python
El código que sigue los principios de Python es reconocido como **«pythónico»**. Estos principios, descritos por Tim Peters, resumen la filosofía del lenguaje:

| Principio | Concepto |
| :--- | :--- |
| **Bello** es mejor que feo | La estética del código importa. |
| **Explícito** es mejor que implícito | No dejes cosas a la imaginación del intérprete. |
| **Simple** es mejor que complejo | Evita la sobreingeniería. |
| **La legibilidad** cuenta | Escribimos código para que otros humanos lo entiendan. |
| **Lo práctico** gana a lo puro | El lenguaje debe resolver problemas reales. |

> *"Frente a la ambigüedad, rechaza la tentación de adivinar."*
